# Projet Santé — Système de Gestion des Indicateurs Sanitaires

## Description du Projet
Ce projet développe un système de gestion et de suivi des indicateurs de santé publique pour le système de santé congolais. Il centralise les données des hôpitaux, calcule les KPIs selon les normes OMS et génère des rapports pour les décideurs.

## Objectifs
- Centraliser les données des établissements de santé
- Calculer automatiquement les indicateurs OMS (densité médicale, mortalité, occupation)
- Générer des rapports comparatifs pour le Ministère de la Santé
- Identifier les situations critiques et alerter en temps réel
